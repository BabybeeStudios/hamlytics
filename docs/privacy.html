Privacy Policy

Effective date: 2026-01-20

Hamlytics is a Chrome extension that analyzes publicly available TikTok profile information at the user’s request.

Data Collection

Hamlytics does not collect, sell, or share personal user data.

  • No personally identifiable information is collected.
  • No authentication credentials or passwords are collected.
  • No financial or payment data is collected by the extension.
  • No location data is collected.
  • No browsing history is collected beyond the active TikTok profile page the user chooses to scan.

How the Extension Works

Hamlytics reads publicly visible data from the TikTok profile page the user is viewing when the user clicks “Scan.” All processing occurs locally in the browser.

Storage

The extension may store settings (such as plan status or a Pro token) using Chrome local storage. This data remains on the user’s device.

Payments

Pro purchases are handled externally via Stripe on the Hamlytics website. The extension itself does not collect payment information.
