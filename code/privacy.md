Hamlytics Privacy Policy

Last updated: January 2026

Hamlytics is a Chrome extension that helps users analyze publicly available TikTok profile data.

Information We Collect
Hamlytics collects only the minimum data required to provide its functionality:
	•	Website content: Publicly available TikTok profile information (such as video URLs and public metrics like views, likes/comments when Pro Deep Scan is enabled).
	•	Authentication information: A Pro access token, if the user chooses to activate Hamlytics Pro. This token is stored locally in the user’s browser.

Information We Do Not Collect
Hamlytics does not collect:
	•	Names, email addresses, or other personally identifiable information
	•	Health data
	•	Financial or payment information
	•	Location data
	•	Personal communications
	•	Web browsing history
	•	Keystrokes or behavioral tracking

Data Usage
	•	All TikTok profile analysis is performed locally in the user’s browser.
	•	The Pro token is used only to verify feature access and is not shared with third parties.

Data Sharing
Hamlytics does not sell, rent, or share user data with third parties.

Data Storage
	•	Any stored data (such as the Pro token) is saved locally using Chrome’s storage APIs.
	•	Users may remove all stored data at any time by uninstalling the extension.

Changes
This policy may be updated as features evolve. Any changes will be reflected on this page.
